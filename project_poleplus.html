Project Brief:

Poleplus is a pole dancing studio which offers a new way to get fit and have fun while doing it. This company had a need to create an online identity with a booking system for the classes offered.

I created a custom booking system in PHP and MySQL which allows the creation and management of classes offered and from a customer point of view walked them through the booking with an email sent to the customer confirming the booking.

I enjoyed working with the client to create a custom solution for their needs.

Another noteable feature in this project was the facebook integration for page likes etc...
