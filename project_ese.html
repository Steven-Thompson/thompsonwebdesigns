Project Brief:

This project was a university assignment for a unit which focused on interactivity design. I elected to create a lifestyle assessment website which will give advise on areas to improve your health.

For this project I wanted to create something I have never tried before which is a single page parallax website. I decided to use Skrollr and Waypoints JS for the scrolling and to display different elements as you scroll down the page.

This project was good to learn and understand jQuery. I wrote some jQuery code to check all of the form fields on the website and generate an animation to display the end result to the user. If one of the fields were missing a different message was generated.
